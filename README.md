# Advanced Excel Business Analytics Workbook

## Overview

This project is an Advanced Excel workbook that brings together multiple business-style datasets and exercises to showcase core analytics skills: data cleaning, formulas, logical operations, lookups, pivot tables, consolidation, and basic reporting.[file:37]

The goal is to demonstrate that I can use Excel not just as a spreadsheet tool, but as a practical business analytics environment to clean data, build calculations, and create simple summaries and dashboards.

## Skills Demonstrated

The workbook contains multiple sheets, each focused on specific Excel capabilities and business scenarios:[file:37]

- **Data cleaning**
  - Using Freeze Panes and Text to Columns.
  - Cleaning and standardizing names, addresses, emails, and phone numbers.
  - Handling errors such as `#DIV/0!` in KPI calculations.

- **Formulas and logical operations**
  - Basic aggregations: total sales, average sale price, maximum, minimum.
  - IF and nested IF functions to classify pass/fail, assign bonuses, and accept/reject projects.
  - Advanced logical conditions for business rules (e.g., revenue thresholds, profit cut-offs).

- **Lookup and reference**
  - VLOOKUP and HLOOKUP to pull employee, product, and inventory information from master tables.
  - Consolidating data across multiple divisions (e.g., regional summaries for New Hampshire, Maine, Connecticut).

- **Pivot tables and charts**
  - Pivot table summarizing total revenue and average profit by product.
  - Pivot-based charts to quickly visualize performance by product or category.

- **Conditional formatting**
  - Highlighting the lowest or highest orders using top/bottom rules.
  - Highlighting records that meet certain criteria (e.g., outliers, underperforming items).

- **Business metrics**
  - Revenue, cost, profit, and profit margin over time.
  - Phone calls → leads → sales conversion metrics.
  - Project evaluation (ACCEPT / REJECT) based on minimum revenue and profit requirements.
  - Bonus calculations for employees exceeding revenue targets.

## Example Sheets

Some representative sheets in the workbook:[file:37]

- **Sales Summary:** Calculates total sales, average sale value, maximum and minimum order using a list of transactions.
- **Student Performance:** Uses IF and nested logic to assign pass/fail, letter grades, and awards to top performers.
- **Revenue & Profit Margin:** Calculates revenue changes, profit, and profit margins day by day.
- **Product Revenue & Profit (Pivot):** Uses a pivot table to summarize total revenue and average profit by product and supports charting.
- **Project Selection:** Uses IF logic to accept or reject projects based on minimum revenue and profit thresholds.
- **Employee & Inventory Lookups:** Demonstrates VLOOKUP/HLOOKUP to fetch employee details and inventory status from master lists.
- **Consolidated Division Summary:** Aggregates quarterly spending or revenue across multiple divisions into a single consolidated view.

## How to Use

1. Download the workbook:
   - From GitHub: click on `advanced_excel_business_analytics.xlsx` and choose “Download”.
2. Open it in Microsoft Excel or a compatible spreadsheet application.
3. Explore sheets one by one to see:
   - Formulas in the formula bar.
   - How logical tests and lookups are set up.
   - How pivot tables summarize data.
   - How conditional formatting is applied to highlight key records.
4. Modify the data or formulas to experiment with different scenarios (e.g., change thresholds, adjust bonus rules).

## Why This Project Matters

For business analytics and data analytics roles, Excel is still a core tool in day-to-day work. This workbook shows:

- Comfort with real-world style datasets (sales, projects, employees, inventory).
- Ability to translate business rules into formulas and logical conditions.
- Ability to summarize data using pivot tables, lookups, and basic visualizations.
- Good foundation to build dashboards or feed more advanced tools like Power BI and Python.
